# 🧠 艾宾浩斯记忆曲线复习功能

## 功能概述

基于艾宾浩斯遗忘曲线原理，实现科学的单词复习系统：
- 错误的词汇自动加入复习计划
- 按照记忆曲线在特定时间间隔复习
- 连续答对2次后移除，表示已掌握

## 核心特性

### 1. **智能复习时间表**
根据艾宾浩斯遗忘曲线，在以下时间间隔复习：
- 第1阶段：1天后
- 第2阶段：2天后
- 第3阶段：4天后
- 第4阶段：7天后
- 第5阶段：15天后

### 2. **掌握标准**
- 连续答对 **2次** 后，词汇从复习列表移除
- 如果答错，重置连续答对次数，从第1阶段重新开始

### 3. **优先级系统**
听写题目选择优先级：
1. 🔴 **到期需复习**（最多占50%）：已到复习时间的词汇
2. 🟡 **待复习**（最多占30%）：还未到复习时间的词汇
3. ⚪ **正常词库**（剩余）：从词库随机选择

### 4. **进度追踪**
- 实时显示复习阶段：如 `[复习阶段2, 1/2次]`
- 🔴 到期复习：已到复习时间
- 🟡 待复习：还未到复习时间
- 答对后显示下次复习时间

## 使用说明

### 自动触发
- **答错词汇**：自动加入复习计划，1天后首次复习
- **答对复习词汇**：
  - 连续答对 < 2次：进入下一复习阶段
  - 连续答对 = 2次：移除，表示已掌握 🎉

### 查看复习进度
在「📋 错误记录 & 艾宾浩斯复习」标签页查看：
- 📊 各阶段词汇分布
- 📈 答对进度统计
- 🔴 到期需复习列表
- 🟡 待复习列表

## 技术实现

### 数据结构
```json
{
  "english_word": {
    "chinese": "中文释义",
    "type": "word/sentence",
    "added_time": "2026-04-10 19:00:00",
    "ebbinghaus_stage": 0,          // 当前复习阶段 (0-4)
    "next_review_time": "2026-04-11 00:00:00",  // 下次复习时间
    "consecutive_correct": 0,        // 连续答对次数
    "review_history": [              // 复习历史
      {
        "time": "2026-04-11 10:00:00",
        "result": "correct",
        "stage": 0
      }
    ]
  }
}
```

### 核心函数
- `add_to_review_list()`: 添加到复习列表，初始化复习计划
- `update_review_progress()`: 更新复习进度，处理答对逻辑
- `get_due_review_items()`: 获取到期需复习的词汇
- `get_review_statistics()`: 获取复习统计信息
- `select_items_for_quiz()`: 智能选择听写题目（优先到期复习）

## 配置参数

在 `app.py` 顶部可以修改：
```python
# 复习时间间隔（天）
EBBINGHAUS_INTERVALS = [1, 2, 4, 7, 15]

# 连续答对次数达标后移除
CORRECT_THRESHOLD = 2
```

## 优势

1. **科学依据**：基于心理学研究的艾宾浩斯遗忘曲线
2. **自动化**：无需手动管理，系统自动安排复习
3. **可视化**：清晰展示复习进度和阶段
4. **灵活性**：可调整复习间隔和掌握标准
5. **激励性**：连续答对2次即可掌握，给用户成就感
