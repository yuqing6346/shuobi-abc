#### 2. 🎯 智能学习机制 - 一次掌握，不再重复
实现真正的智能学习系统：

**核心规则**：
- ✅ 如果单词**第一次就答对（无提示）**，该单词会被标记为"已掌握"
- ✅ **已掌握的单词不会再出现在正常听写中**
- ✅ 如果后来该单词被答错，会重新加入复习列表，直到再次掌握
- ✅ 在"📊 成就统计"中可以查看已掌握单词数量

**学习效果**：
- 避免重复练习已经会的单词
- 集中精力在不熟悉的词汇上
- 提高学习效率和成就感
- 鼓励孩子不依赖提示，一次性答对

**反馈提示**：
```
✅ 正确！ +10分 📚已掌握！
```
当看到"📚已掌握"时，说明这个单词已经真正学会了！

## 🧠 [v2.1] - 2026-04-10

### 🆕 新增功能：艾宾浩斯记忆曲线复习系统

#### 1. 📚 科学复习机制
基于艾宾浩斯遗忘曲线，实现智能复习系统：
- **自动化**：答错的词汇自动加入复习计划
- **科学时间间隔**：在 **1天、2天、4天、7天、15天** 后复习
- **掌握标准**：连续答对 **2次** 即可从复习列表移除
- **进度追踪**：实时显示复习阶段和答对次数

#### 2. 🎯 智能题目选择
优先级系统确保高效复习：
1. 🔴 **到期需复习**（最多占50%）：已到复习时间的词汇
2. 🟡 **待复习**（最多占30%）：还未到复习时间的词汇
3. ⚪ **正常词库**（剩余）：从词库随机选择

#### 3. 📊 复习统计面板
在「📋 错误记录 & 艾宾浩斯复习」标签页显示：
- **总览统计**
  - 总复习词汇数量
  - 到期需复习数量
  - 复习原理说明
  - 掌握标准

- **阶段分布**
  - 第1阶段(1天)：X 个
  - 第2阶段(2天)：X 个
  - 第3阶段(4天)：X 个
  - 第4阶段(7天)：X 个
  - 第5阶段(15天)：X 个

- **答对进度**
  - 连续答对 0/2 次：X 个
  - 连续答对 1/2 次：X 个

#### 4. 🎮 实时反馈增强
**听写界面显示**：
```
🎧 请听音频写出 📝 单词 🔴 到期复习 [复习阶段1, 0/2次] (历史错误: 3次)
```

**答对反馈**：
- 进行中：`✅ 正确！ +10分 ✅ 答对 1/2 次 下次复习: 2天后`
- 掌握：`✅ 正确！ +10分 🎉 已掌握！连续答对2次 已从复习列表移除！`

**答错处理**：
- 重置连续答对次数
- 回到第1阶段（1天后复习）
- 保留在复习列表中

#### 5. 📝 数据结构
新增 `review_list.json` 字段：
```json
{
  "english_word": {
    "chinese": "中文释义",
    "type": "word/sentence",
    "added_time": "2026-04-10 19:00:00",
    "ebbinghaus_stage": 0,          // 当前复习阶段 (0-4)
    "next_review_time": "2026-04-11 00:00:00",  // 下次复习时间
    "consecutive_correct": 0,        // 连续答对次数
    "review_history": []             // 复习历史记录
  }
}
```

### 🔧 核心函数
- `add_to_review_list()` - 初始化复习计划
- `update_review_progress()` - 更新复习进度，处理答对逻辑
- `get_due_review_items()` - 获取到期词汇

## 🎉 [v2.0] - 2026-04-10

### 🆕 新增功能：成就统计系统

#### 1. 💰 积分体系
- **基础分数**
  - 单词无提示答对：10分
  - 句子无提示答对：20分
  - 使用提示答对：0分（不获得积分）

- **连击加分机制**
  - 3连击：基础分数 + 50%额外加分
  - 5连击：基础分数 + 100%额外加分
  - 7连击：基础分数 + 150%额外加分
  - 10连击：基础分数 + 200%额外加分

- **示例**
  - 单词无提示答对（3连击）：10 + 5 = 15分
  - 句子无提示答对（5连击）：20 + 20 = 40分
  - 句子无提示答对（10连击）：20 + 40 = 60分

#### 2. 🏅 Minecraft风格称号系统
10个等级称号，每1000分晋升一级：

| 积分范围 | 称号 | 说明 |
|---------|------|------|
| 0-999 | 🌱 新手冒险家 | 刚开始探险的小萌新 |
| 1000-1999 | ⛏️ 石器工匠 | 掌握了基本工具 |
| 2000-2999 | ⚔️ 铁甲战士 | 装备齐全的战士 |
| 3000-3999 | 🏅 金牌猎人 | 经验丰富的猎手 |
| 4000-4999 | 💎 钻石骑士 | 拥有钻石装备 |
| 5000-5999 | 💚 绿宝石商人 | 富有的商人 |
| 6000-6999 | 🔴 红石工程师 | 精通红石科技 |
| 7000-7999 | 🔥 下界探索者 | 敢于探索下界 |
| 8000-9999 | 🐉 末影龙骑士 | 战胜末影龙 |
| 10000+ | 👑 创世神 | 最高荣誉！ |

#### 3. 📊 听写轮次统计
- **完成轮次**：记录总共完成的听写轮次
- **达成天数**：记录完成听写的不同日期数量
- 每完成一轮听写，轮次+1，且当天会被记录

#### 4. 📚 词汇掌握统计
- **掌握单词**：无提示答对的单词会被永久记录
- **掌握句子**：无提示答对的句子会被永久记录
- 同一个词汇只统计一次
- 使用提示答对不会被计入掌握

#### 5. 🔥 无提示连对记录
- **当前连对**：当前无提示连续答对的题数
- **最高连对**：历史最高无提示连对记录
- 使用提示或答错会中断连对

#### 6. 📈 成就统计页面
新增专门的"📊 成就统计"标签页，显示：
- 当前称号和积分
- 到下一称号还需多少分
- 完成轮次和达成天数
- 掌握的单词/句子数量
- 最高连击和最高无提示连对记录
- 成就解锁进度

#### 7. 🎯 新增成就
- 📝 **初次听写**：完成第一次听写
- 📖 **词汇新手**：掌握10个单词
- 📚 **词汇达人**：掌握50个单词
- 🎓 **词汇大师**：掌握100个单词
- 🌟 **千分之旅**：获得1000积分
- ⭐ **五千之路**：获得5000积分
- 💯 **完美连击**：无提示连对20题
