## 💻 步骤二：新电脑准备环境

### 1️⃣ 安装 Python

#### 检查是否已安装
打开 CMD 或 PowerShell，输入：
```bash
python --version
```

如果显示 `Python 3.8` 或更高版本，可以跳过安装步骤。

#### 如果未安装
1. 访问 Python 官网：https://www.python.org/downloads/
2. 下载 Python 3.9 或 3.10（推荐）
3. 安装时**务必勾选** `Add Python to PATH`
4. 安装完成后重启 CMD 验证

### 2️⃣ 解压项目文件

将打包的文件解压到任意位置，例如：
```
D:\英语听写工具\
```

### 3️⃣ 安装依赖包

#### 打开 CMD 或 PowerShell

1. 按 `Win + R`，输入 `cmd`，回车
2. 切换到项目目录：
```bash
cd /d D:\英语听写工具
```

#### 安装依赖
```bash
pip install -r requirements.txt
```

如果速度很慢，可以使用国内镜像源：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

#### 验证安装
安装完成后，可以验证关键包：
```bash
pip list | findstr gradio
pip list | findstr kittentts
```

## 🔧 常见问题与解决

### ❌ 问题1：python 不是内部或外部命令

**原因**：Python 未安装或未添加到环境变量

**解决**：
1. 重新安装 Python，勾选 `Add Python to PATH`
2. 或手动添加环境变量：`系统属性` → `环境变量` → `Path` → 添加 Python 安装路径

### ❌ 问题2：pip install 失败

**原因**：网络问题或权限不足

**解决**：
```bash
# 使用国内镜像源
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# 如果权限不足，使用管理员权限运行 CMD
# 右键 CMD → 以管理员身份运行
```

### ❌ 问题3：端口 7860 被占用

**解决**：
```bash
# 方法1：修改端口号
# 打开 app.py，找到最后一行，修改为：
app.launch(share=False, server_name="0.0.0.0", server_port=7861, css=custom_css)

# 方法2：关闭占用端口的程序
netstat -ano | findstr :7860
taskkill /F /PID [进程ID]
```

### ❌ 问题4：模型下载失败

**原因**：HuggingFace 连接超时

**解决**：
- 已配置 HuggingFace 镜像源，会自动重试
- 如果多次失败，检查网络连接
- 稍后重试：关闭程序，重新启动

### ❌ 问题5：没有数据文件

**原因**：首次启动，没有复制旧数据

**解决**：
- 这是正常的，程序会自动创建新文件
- 如果想要旧数据，从原电脑复制对应的 .json 文件

## 💡 推荐传输方式

### 方式1：U盘传输（推荐）
1. 复制打包文件夹到 U 盘
2. 在新电脑上从 U 盘复制到本地硬盘
3. 按照步骤安装依赖并启动

### 方式2：云盘传输
1. 上传到百度网盘/OneDrive 等
2. 在新电脑下载
3. 解压后按步骤操作

### 方式3：局域网传输
如果两台电脑在同一网络：
1. 使用共享文件夹
2. 或使用 QQ/微信文件传输
