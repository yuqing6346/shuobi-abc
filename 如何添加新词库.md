# 📚 如何添加新词库（五年级、六年级、KET等）

## 快速添加词库步骤

### 1️⃣ 打开 `word_bank.py` 文件

### 2️⃣ 在 `WORD_BANK` 字典中添加新词库

在 `WORD_BANK` 字典的最后，添加新的年级词库。格式如下：

```python
"五年级": {
    "words": [
        {"english": "单词1", "chinese": "中文意思1"},
        {"english": "单词2", "chinese": "中文意思2"},
        # 添加更多单词...
    ],
    "sentences": [
        {"english": "句子1", "chinese": "中文翻译1"},
        {"english": "句子2", "chinese": "中文翻译2"},
        # 添加更多句子...
    ]
},
```

## 📝 注意事项

1. **格式要求**：
   - 每个词库必须包含 `"words"` 和 `"sentences"` 两个列表
   - 每个单词/句子必须包含 `"english"` 和 `"chinese"` 字段
   - 注意中文逗号和英文逗号的区别，Python代码中用英文逗号

2. **命名规范**：
   - 词库名称可以是：`"一年级"`、`"二年级"`、`"五年级"`、`"六年级"`、`"KET"`、`"PET"`等
   - 名称会自动出现在界面的词库选择下拉框中

3. **保存位置**：
   - 找到 `word_bank.py` 文件中的 `WORD_BANK = {` 
   - 在最后一个词库（目前是"四年级"）后面添加逗号
   - 然后添加新的词库

4. **添加后的效果**：
   - 新词库会自动出现在界面的"📚 选择词库"复选框中
   - 选中后会自动统计单词数和句子数
   - 可以与其他词库组合使用

## 💡 提示

- 如果单词或句子较多，建议按照教材的单元（Unit）进行注释分组，便于管理
- 可以先添加少量词汇测试，确认无误后再批量添加
- 添加完成后，建议测试一下听写功能是否正常
