# 📦 打包版本更新指南

适用于家里使用**打包版本**的用户。

---

## 🎯 两种更新方式

### 方式1：使用更新工具（推荐）⭐

**步骤：**

1. **下载最新代码**
   - 访问：https://github.com/yuqing6346/shuobi-abc
   - 点击绿色 **Code** 按钮 → **Download ZIP**
   - 解压到任意位置（比如桌面）

2. **运行更新工具**
   - 在打包文件夹中双击 `打包版本更新工具.bat`
   - 按提示输入新代码文件夹路径（或直接拖拽文件夹到窗口）
   - 确认更新

3. **完成！**
   - ✅ 代码已更新
   - ✅ 学习数据完整保留
   - ✅ 旧代码已自动备份

---

### 方式2：手动更新

**步骤：**

1. **备份学习数据**
   ```
   复制这些文件到安全位置：
   - game_save.json
   - review_list.json
   - error_records.json
   ```

2. **下载新代码**
   - 从GitHub下载最新的 `app.py`、`word_bank.py`、`game_state.py`
   - 覆盖打包文件夹中的同名文件

3. **恢复学习数据**
   - 将第1步备份的文件复制回来

4. **更新依赖（可选）**
   ```bash
   pip install -r requirements.txt
   ```

## 💡 提示

### 如果有Git环境
可以把打包文件夹转换为Git仓库：
```bash
cd "打包文件夹路径"
git init
git remote add origin https://github.com/yuqing6346/shuobi-abc.git
git fetch
git reset --hard origin/main
# 数据文件会被.gitignore保护，不会被覆盖
```

之后就可以使用 `智能更新_保留数据.bat` 一键更新了！

### 检查更新内容
访问：https://github.com/yuqing6346/shuobi-abc/commits/main
查看最新的代码提交记录

## ❓ 常见问题

**Q: 更新后数据会丢失吗？**
A: 不会！使用更新工具会自动保留所有学习数据

**Q: 更新失败怎么办？**
A: 旧代码会自动备份在 `_code_backup_日期时间` 文件夹中，可以手动恢复

**Q: 需要重新安装依赖吗？**
A: 大多数情况不需要。如果更新后运行出错，运行：`pip install -r requirements.txt`

**Q: 可以回退到旧版本吗？**
A: 可以！备份文件夹中有旧代码，直接复制回来即可
