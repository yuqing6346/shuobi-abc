## 📍 当前电脑（打包）

### 方法1：自动打包（推荐）⭐

1. **双击运行** `打包脚本.bat`
2. 等待完成，会自动创建 `英语听写工具_打包_日期` 文件夹
3. 将该文件夹复制到 U 盘或压缩成 ZIP

### 方法2：手动打包

复制这些文件到一个新文件夹：
- ✅ `app.py`
- ✅ `word_bank.py`
- ✅ `game_state.py`
- ✅ `requirements.txt`
- ✅ `README.md`
- 🔄 `game_save.json`（如果想保留数据）
- 🔄 `error_records.json`（可选）
- 🔄 `review_list.json`（可选）

## 💻 新电脑（部署）

### 第一步：安装 Python

1. 访问：https://www.python.org/downloads/
2. 下载 Python 3.9 或 3.10
3. 安装时 **勾选** `Add Python to PATH` ⚠️

### 第二步：解压项目

将打包文件解压到任意位置，例如：
```
D:\英语听写工具\
```

### 第三步：安装依赖

打开 CMD（Win + R，输入 cmd），执行：

```bash
# 进入项目目录
cd /d D:\英语听写工具

# 安装依赖（使用国内镜像，速度快）
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 第四步：启动项目

**方法1**：双击 `启动.bat`（如果有这个文件）

**方法2**：在 CMD 中运行：
```bash
python app.py
```

### 第五步：访问应用

打开浏览器，访问：
```
http://localhost:7860
```

## ⚠️ 常见问题

### ❌ python 不是内部或外部命令
👉 重新安装 Python，勾选 `Add Python to PATH`

### ❌ pip install 很慢或失败
👉 使用国内镜像：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### ❌ 端口 7860 被占用
👉 关闭其他程序或修改端口号（编辑 app.py 最后一行）

### ❌ 模型下载失败
👉 等待自动重试，或检查网络连接后重新启动
