# ⛏️ 英语探险家 - Minecraft 听写冒险

> 把英语听写变成 Minecraft 风格的冒险游戏！专为学习动力不足的孩子设计。

## 🎮 游戏化特色

### 核心玩法
- **📦 收集方块** - 每答对一题就能获得方块（单词=普通方块，句子=高级方块）
- **🔥 连击奖励** - 连续答对3/5/10题获得钻石等稀有方块
- **🐾 驯服怪物** - 答错的单词变成怪物，再次答对可以驯服获得双倍奖励
- **🏗️ 建造建筑** - 累积足够方块自动解锁：营火 → 小木屋 → 城堡 → 钻石宫殿...
- **🏆 成就系统** - 各种里程碑成就等你解锁
- **🎁 每日奖励** - 每天登录都有惊喜宝箱！

### 🆕 新增：成就统计系统
- **💰 积分体系** - 单词10分，句子20分，连击额外加分
- **🏅 称号系统** - 从🌱新手冒险家到👑创世神，共10个Minecraft风格称号
- **📊 轮次统计** - 记录完成听写的轮次和达成天数
- **📚 词汇掌握** - 追踪无提示答对的单词/句子数量
- **🔥 连对记录** - 最高无提示连对次数统计

### 针对学习动力不足的设计
- ✅ **低门槛** - 新手村模式90%都是简单单词
- ✅ **高频奖励** - 每答对一题立刻获得奖励
- ✅ **避免挫败感** - 错误不是失败，而是"发现新怪物"
- ✅ **成就可视化** - 建筑进度让学习成果看得见

## 🚀 快速开始

### 安装依赖
```bash
pip install -r requirements.txt
```

### 运行游戏化版本（推荐）
```bash
python app_game.py
```

### 运行原版（无游戏化）
```bash
python app.py
```

访问 http://127.0.0.1:7860 开始冒险！

**积分获取规则：**
- 单词无提示答对：10分
- 句子无提示答对：20分
- 3连击：额外50%加分
- 5连击：额外100%加分
- 7连击：额外150%加分
- 10连击：额外200%加分
- ⚠️ 使用提示答对：不获得积分

### 成就系统
- 🎉 第一块方块
- ⚔️ 怪物猎人（驯服第一只怪物）
- 🔥 连击新手/达人/大师（3/5/10连击）
- 🏗️ 建造者（解锁第一个建筑）
- 💎 钻石猎人
- 📅 坚持三天/一周达人
- 📦 收藏家（50/100方块）
- 👹 怪物大师（驯服所有种类怪物）
- 🆕 📝 初次听写（完成第一次听写）
- 🆕 📖 词汇新手/达人/大师（掌握10/50/100个单词）
- 🆕 🌟 千分之旅/五千之路（获得1000/5000积分）
- 🆕 💯 完美连击（无提示连对20题）
