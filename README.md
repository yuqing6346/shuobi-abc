# 🎯 英语单词听写工具

一个基于AI语音合成的英语听写学习工具，支持艾宾浩斯记忆曲线复习。

## ✨ 功能特点

- 🎯 **多词库支持**：小学-高中各年级词库
- 🔊 **AI真人发音**：KittenTTS语音合成
- 📊 **智能错题记录**：自动记录易错单词
- 🧠 **艾宾浩斯记忆曲线**：科学复习计划
- 📈 **学习进度追踪**：实时统计学习数据
- 🎮 **游戏化学习**：趣味答题模式

## 🚀 使用方法

直接在上方界面开始学习，无需安装！

## 📝 提示

- 首次加载需要下载语音模型，请耐心等待
- 建议使用电脑浏览器访问以获得最佳体验
- 学习数据仅保存在浏览器会话中
