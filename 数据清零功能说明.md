# 🔄 数据清零功能使用说明

## 📅 更新日期
2026年4月11日

## ✨ 新增功能

### 🔄 一键清零所有数据

在"📋 错误记录 & 艾宾浩斯复习"标签页底部，新增了**数据重置功能**。

## 📍 功能位置

1. 打开应用程序
2. 点击顶部的"**📋 错误记录 & 艾宾浩斯复习**"标签
3. 滚动到页面底部
4. 找到"**⚠️ 数据重置**"区域
5. 点击"**🔄 重置所有数据（归零）**"按钮

## 🗑️ 会清除的数据

点击清零按钮后，以下所有数据将被清空：

### 游戏数据
- ✅ 积分（Score）归零
- ✅ 称号重置为"新手探险家"
- ✅ 方块背包清空
- ✅ 怪物图鉴清空
- ✅ 建筑进度清空
- ✅ 成就清空
- ✅ 等级和经验重置

### 学习数据
- ✅ 掌握的单词列表清空
- ✅ 掌握的句子列表清空
- ✅ 连击记录归零
- ✅ 听写轮次统计归零
- ✅ 错误记录清空
- ✅ 复习列表清空（艾宾浩斯复习计划）

## 🗂️ 数据备份

**重要**：在清零之前，系统会自动备份你的旧数据！

- 备份文件名格式：`game_save_backup_20260411_123045.json`
- 备份位置：项目根目录 `d:\cursorRepo\abc\`
- 如果误操作，可以手动恢复：将备份文件重命名为 `game_save.json`

## 📋 操作步骤

### 步骤1：打开清零页面
```
应用主界面 → 📋 错误记录 & 艾宾浩斯复习 → 滚动到底部
```

### 步骤2：点击清零按钮
```
点击 "🔄 重置所有数据（归零）" 按钮（红色）
```

### 步骤3：确认清零成功
```
看到成功提示：
✅ 重置成功！
所有数据已清零...
🗂️ 旧数据已自动备份到项目文件夹
```

### 步骤4：刷新页面
```
按 F5 或点击浏览器刷新按钮
```

## 🔄 刷新后的效果

刷新页面后，你会看到：
- 🎖️ 称号：🌱 新手探险家
- 📊 当前积分：0 分
- 📝 单词掌握：0 个
- 📄 句子掌握：0 个
- 📅 累计听写：0 天

一切从头开始！🎉

## ⚠️ 注意事项

1. **不可逆操作**：清零后无法直接恢复（除非手动恢复备份文件）
2. **测试数据**：适合用于清除测试数据，重新开始正式学习
3. **备份安全**：请保管好自动生成的备份文件
4. **需刷新页面**：清零后需要刷新浏览器页面才能看到效果

## 🔧 手动恢复备份

如果误操作想要恢复，可以这样做：

### 方法1：直接替换
1. 关闭应用程序
2. 找到备份文件（如 `game_save_backup_20260411_123045.json`）
3. 删除 `game_save.json`
4. 将备份文件重命名为 `game_save.json`
5. 重启应用程序

### 方法2：复制内容
1. 用记事本打开备份文件
2. 复制全部内容
3. 打开 `game_save.json`
4. 粘贴并保存
5. 重启应用程序

## 💡 使用场景

### 场景1：清除测试数据
在正式开始学习前，清除所有测试数据，从零开始。

### 场景2：重新开始
想要重新开始听写学习，清空所有历史记录。

### 场景3：多人使用
如果多人使用同一个账号，可以在切换用户时清零。

### 场景4：统计重置
想要重新统计学习效果，清空旧数据。
